Lab Objectives

  • Master Matplotlib for basic plotting (Line, Scatter, Bar).
  • Use Seaborn for statistical visualization (Distribution, Boxplots).
  • Understand EDA (Exploratory Data Analysis) to find patterns.
  • Visualize correlations using Heatmaps.

Part 1: Pandas Non-Visual EDA

Q1

Inspection & Structure

Load the Iris dataset. Display the first 5 rows, the data types of columns, and the total memory usage of the DataFrame.

Q2

Statistical Summary

Use a single Pandas command to get the count, mean, std, min, and quartiles for all numerical columns. Transpose the result for better readability.

Q3

Class Balance

Check if the dataset is balanced. Count the number of samples for each `species`. Also, check for any null values in the dataset.

Q4

Aggregations

Group the data by `species` and calculate the mean and median of `sepal_length` for each species side-by-side.

Part 2: NumPy Analysis

Q5

Array Conversion

Extract the `sepal_length` column as a NumPy array. Calculate its variance and standard deviation using NumPy functions.

Q6

Standardization (Z-Score)

Standardize the `sepal_width` column using NumPy: `(x - mean) / std`. Create a new column `sepal_width_z` in the dataframe with these values.

Q7

Filtering with NumPy

Use np.where to create a new column `petal_size_category`. If `petal_length` > 4.5, assign 'Large', otherwise 'Small'.

Q8

Matrix Operations

Compute the Correlation Matrix using NumPy (`np.corrcoef`) on the numerical features only. Compare it with Pandas `.corr()`.

Part 3: Matplotlib Visualization

Q9

Scatter Plot

Create a scatter plot of `sepal_length` vs `sepal_width`. Label axes clearly and add a title "Sepal Dimensions".

Q10

Histograms

Plot a histogram of `petal_length` with 15 bins. Set the color to green and transparency (alpha) to 0.7.

Part 4: Advanced Seaborn EDA

Q11

Boxplot (Outlier Detection)

Create a boxplot comparing `petal_width` across different `species`. Identify which species has outliers.

Q12

Violin Plot

Generate a violin plot for `sepal_length` by `species`. How does this compare to the boxplot information?

Q13

Pairplot (Multivariate)

Use `sns.pairplot` to visualize relationships between all numerical variables, distinguishing species by color (hue).

Q14

Heatmap (Correlation)

Compute the correlation matrix and visualize it using a heatmap with annotations. Which two features are most strongly positively correlated?

Q15

FacetGrid

Use `sns.FacetGrid` to create a grid of histograms for `sepal_length`, with one plot for each `species`.

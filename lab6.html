Q1 - State Space Search

8-Puzzle Problem

Hard

You are given a 3 x 3 board representing the 8-puzzle. The board contains tiles numbered from 1 to 8 and one empty space represented by 0.

In a single move, you can slide a tile adjacent to the empty space (horizontally or vertically) into the empty space. Your goal is to write a program that finds the minimum number of moves required to transform the initial state of the board into the target state.

Example 1:
Initial
1 2 3
4 0 5
7 8 6
Target
1 2 3
4 5 6
7 8 0
Input:
initial = [[1, 2, 3], [4, 0, 5], [7, 8, 6]]
target = [[1, 2, 3], [4, 5, 6], [7, 8, 0]]
Output: 2
Explanation:
Move 1: Slide the '5' left into the empty space.
Move 2: Slide the '6' up into the empty space.
Example 2:
Input:
initial = [[1, 2, 3], [4, 5, 6], [7, 8, 0]]
target = [[1, 2, 3], [4, 5, 6], [7, 8, 0]]
Output: 0
Explanation: The board is already in the target state.

Constraints:

  • initial.length == target.length == 3
  • initial[i].length == target[i].length == 3
  • Both boards contain the integers 0 through 8 exactly once.
  • Assume a solution always exists for the given test cases.
Q2 - Backtracking

4-Queens Simulation

Intermediate

The 4-queens puzzle is the problem of placing 4 chess queens on a 4 x 4 chessboard such that no two queens threaten each other.

This means no two queens can share the same row, column, or diagonal. Write a program to return all distinct solutions to the 4-queens puzzle. Each solution should be a list of strings where 'Q' and '.' indicate a queen and an empty space, respectively.

Example 1:
Input: n = 4
Output:
[
  [".Q..", // Solution 1
   "...Q",
   "Q...",
   "..Q."],

  ["..Q.", // Solution 2
   "Q...",
   "...Q",
   ".Q.."]
]
Explanation: There exist exactly two distinct solutions to the 4-queens puzzle as shown above.

Constraints:

  • n == 4
  • Your algorithm should use a recursive backtracking approach to simulate placing the queens row by row.

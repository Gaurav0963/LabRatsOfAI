Lab 2

Data Manipulation with Pandas

Date: Jan 19, 2026 Subject: AI Lab
IIIT Bhagalpur
Dept. of Computer Science

Required Dataset: Titanic

This lab requires the Titanic Passenger dataset. Download it before starting.

Download .csv

Lab Objectives

  • Load and inspect tabular data using DataFrames.
  • Handle missing values (imputation vs. dropping).
  • Perform advanced filtering, sorting, and indexing.
  • Master groupby, aggregations, and pivot tables.

Part 1: Loading & Inspection

Q1

Load Data

Load the titanic.csv file into a Pandas DataFrame. Display the first 5 rows, check the column data types (`.info()`), and generate statistical summaries (`.describe()`).

Q2

Column Selection

Create a new DataFrame containing only the Name, Sex, and Age columns. Display its shape.

Q3

Sorting

Sort the dataset by Fare in descending order. Who paid the highest fare? Display their name.

Part 2: Filtering & Indexing

Q4

Conditional Filtering

Filter the data to find all Female passengers who survived (Survived = 1). How many such passengers are there?

Q5

Complex Conditions

Find all passengers who are in First Class (Pclass = 1) AND are older than 50. Display their names and ages.

Q6

Loc vs Iloc

Use .iloc to select rows 10 to 20 and columns 0 to 3. Then use .loc to select the 'Name' of the passenger with index 100.

Part 3: Handling Missing Values

Q7

Null Detection

Check which columns have missing values (NaN). Count the number of missing values in each column.

Q8

Imputation

The Age column has missing values. Fill these missing values with the median age of the dataset. Verify that no nulls remain in Age.

Q9

Dropping Columns

The Cabin column has too many missing values. Drop this column entirely from the DataFrame.

Part 4: GroupBy & Aggregation

Q10

Group Averages

Group the data by Pclass. Calculate the average Fare and average Age for each class.

Q11

Survival Analysis

Group by Sex and calculate the survival rate (mean of the Survived column) for men vs women.

Q12

Pivot Tables

Create a Pivot Table that shows the average survival rate indexed by Sex and columns as Pclass.

Part 5: Feature Engineering

Q13

Creating Columns

Create a new column FamilySize which is the sum of SibSp (Siblings/Spouse) + Parch (Parents/Children) + 1 (the passenger themselves).

Q14

Map & Encode

Convert the Sex column into numbers. Map 'male' to 0 and 'female' to 1. Save this in a new column Sex_Code.

Q15

String Operations

The Name column contains titles (Mr., Mrs., etc.). Create a new column Title by extracting the word before the period (.) in the name.

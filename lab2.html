Week 2 - Lab 2

NumPy Library Mastery

Date: Jan 20, 2026 Subject: AI Lab
IIIT Bhagalpur
Dept. of Computer Science

Lab Objectives

By the end of this session, you should be able to:

  • Create and inspect NumPy arrays of various dimensions.
  • Perform advanced slicing, indexing, and boolean masking.
  • Understand array shapes, reshaping, and broadcasting rules.
  • Execute linear algebra operations and statistical aggregations.

Part 1: Creation & Inspection

Q1

Array Initialization

Create a 1D array of integers from 0 to 9. Then, create a 3x3 matrix of all boolean True values.

Q2

Sequence Generation

Create an array of 10 equally spaced points between 0 and 1 (inclusive) using linspace. Compare this with arange.

Q3

Array Attributes

Create a random 3x4x2 array. Print its ndim (dimensions), shape, size (total elements), and dtype.

Q4

Reshaping

Create a 1D array of numbers from 0 to 19. Reshape it into a matrix of shape (4, 5). Then, flatten it back to 1D.

Part 2: Indexing & Slicing

Q5

Slicing Matrix

Given a 5x5 identity matrix, extract the sub-matrix consisting of the central 3x3 part.

Q6

Boolean Indexing

Create a random array of 20 integers between 1 and 50. Replace all odd numbers in the array with -1 without using a loop.

Q7

Where Condition

Given two arrays a and b, create a new array containing elements from a where a > b, otherwise take elements from b using np.where.

Part 3: Math & Statistics

Q8

Axis Aggregations

Create a 2D array of shape (3, 4). Calculate the sum of columns and the maximum value of rows.

Q9

Broadcasting

Subtract the mean of each row of a matrix from each element of that row. (Hint: Use keepdims=True).

Q10

Standard Deviation

Generate 1000 random numbers from a standard normal distribution. Calculate their mean and standard deviation. Are they close to 0 and 1?

Q11

Matrix Multiplication

Perform matrix multiplication between a (5x3) matrix and a (3x2) matrix. What is the shape of the result?

Part 4: Manipulation & Linear Algebra

Q12

Stacking Arrays

Given two 1D arrays, stack them vertically (`vstack`) to form a 2-row matrix, and horizontally (`hstack`) to form one long array.

Q13

Unique Items & Counts

Generate an array of 50 random integers between 0 and 10. Find the unique values and how often each value appears.

Q14

Linear Algebra (Inverse)

Create a random 3x3 matrix. Compute its determinant and inverse using np.linalg.

Q15

Set Operations

Find the common elements between two arrays: a = np.array([1,2,3,2,3,4,3,4,5,6]) and b = np.array([7,2,10,2,7,4,9,4,9,8]).

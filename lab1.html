Week 1

Python Fundamentals & Logic

Date: Jan 12, 2026 Subject: AI Lab (CS-205)
IIIT Bhagalpur
Dept. of Computer Science

Lab Objectives

  • Master Python's built-in modules (calendar, datetime).
  • Implement conditional logic and control flow.
  • Manipulate strings and iterate using loops.
  • Develop modular code using functions and lists.
Submission Instructions: Create a single Jupyter Notebook (.ipynb) containing solutions for all parts. Use Markdown headers to separate sections (Part 1, Part 2, etc.). Name the file RollNo_Lab01.ipynb.

Part 1: Calendar & Datetime (Experiment 01)

Q1.1

Display a Specific Month

Write a Python program that asks the user for a specific year and month, then displays the calendar for that month using the calendar module.

Q1.2

Display All Months

Write a program to print the calendar for every month of a given year.

Q1.3

Current Date & Time

Write a script to display today's date and the current date and time using the datetime module.

Q1.4

Date Arithmetic

Write a program to take the present date, add a specific number of days to it (e.g., 5 days), and print the resulting future date.

Q1.5

Leap Year Check

Write a program that inputs a year and determines if it is a leap year. (Try doing this both with the calendar module and with simple if-else logic).

Part 2: Conditional Logic (Experiment 02)

Q2.1

Positive, Negative, or Zero

Write a program to read a number from the user and determine if it is positive, negative, or zero.

Q2.2

Largest of Three

Write a program that takes three numbers as input and prints the largest one.

Q2.3

Day of the Week

Write a program that takes a number (1-7) and displays the corresponding day of the week (e.g., 1 = Monday). Handle invalid inputs gracefully.

Q2.4

Simple Calculator

Create a program that takes two numbers and an operator (+, -, *, /) and performs the corresponding calculation.

Part 3: Strings & Loops

Q3.1

Palindrome Check

Write a program to check if a given string is a palindrome (reads the same forwards and backwards).

Q3.2

Fibonacci Series

Write a program to print the first n terms of the Fibonacci sequence using a loop.

Q3.3

Multiplication Table

Write a program to print the multiplication table of a given number (e.g., if input is 5, print 5x1=5, 5x2=10... up to 10).

Part 4: Functions & Lists

Q4.1

Factorial Function

Write a function that calculates the factorial of a given number. Handle the case for negative numbers (where factorial is undefined).

Q4.2

Compare Lists

Write a function that accepts two lists and returns True if they are equal (contain the same elements in the same order), and False otherwise.

Q4.3

List Statistics

Write a program that accepts a list of numbers from the user and calculates the sum, average, and maximum value in that list without using built-in max() or sum() functions.
